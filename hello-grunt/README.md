hello-grunt
==========

1
----------

[Node.js](http://nodejs.org/)をインストールする。

2
----------

	$ npm install -g grunt-cli

として[Grunt](http://gruntjs.com/)のCLIをインストールする。（`sudo`が必要かも。）

3
----------

hello-gruntのディレクトリで

	$ npm install

としてpackage.jsonに記述されている依存パッケージをインストールする。

4
----------

hello-gruntのディレクトリで

	$ grunt

としてGruntを走らせる。
