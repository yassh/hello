hello-gulp
==========

1
----------

[Node.js](http://nodejs.org/)をインストールする。

2
----------

	$ npm install -g gulp

として[gulp](http://gulpjs.com/)をインストールする。（`sudo`が必要かも。）

3
----------

hello-gulpのディレクトリで

	$ npm install

としてpackage.jsonに記述されている依存パッケージをインストールする。

4
----------

hello-gulpのディレクトリで

	$ gulp

としてgulpを走らせる。
